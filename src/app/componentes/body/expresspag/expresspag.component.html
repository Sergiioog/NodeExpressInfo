¿QUÉ ES EXPRESS?

Express es un framework para crear aplicaciones web, APIs y Web Services que ha sido inspirado por la librería de Ruby, Sinastra. Es un software libre, de código abierto que posee licencia del MIT. Fue creado por TJ Holowaychuk y la primera versión fue lanzada el 22 de mayo de 2010. Actualemente, es mantenido por comunidades de desarrolladores.

Una de sus principales características es que proporciona al usuario un conjunto de herramientas para aplicaciones web, peticiones y respuestas HTTP, enrutamiento y middleware para conseguir crear y desplegar applicaciones a gran escala. También proporciona un CLI llamado NPM para que los developers puedan desacargarse paquetes desarrollados. Finalmente, sigue el patrón DRY (Don´t Repeat Yourself) que reduce la repetición de código utilizando normalizaciones de datos para evitar la redundancia.

Express.js se utiliza para una amplia gama de cosas en el ecosistema JavaScript/Node.js puedes desarrollar aplicaciones, endpoints de API, sistemas de enrutamiento y frameworks con él. Ejemplos:SPAS, Netflix/Twitch, Aplicaciones para entidades bancarias etc... En cuánto al RunKit, es un entorno en línea que permite escribir y ejecutar código Node.js directamente en el navegador. Se puede usar para probar rápidamente fragmentos de código, experimentar con bibliotecas y realizar pruebas interactivas.

Hay varias razones por las que Express.js es muu útil, desde una mayor velocidad de E/S para peticiones y respuestas más rápidas hasta su sistema de hilo único y procesos asíncronos. También utiliza la estructura MVC para simplificar las manipulaciones de datos y los sistemas de enrutamiento.
