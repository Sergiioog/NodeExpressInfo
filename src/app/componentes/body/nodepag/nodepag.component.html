¿QUÉ ES NODEJS?

Node JS es un entorno de tiempo de ejecución de JavaScript que se utiliza para crear aplicaciones de lado del servidor a través de servidores privados virtuales

Una de sus principales funciones es asíncrono. Esto quiere decir que se basa en un único hilo, es decir, que sólo puede ejecutar una sola cosa a la vez. Dicha arquitectura está basada en eventos para ayudar a los desarrolladores a crear diversos proyectos de forma eficiente y sencilla
